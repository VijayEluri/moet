DESCRIPTION
    Utility for androidlib.py. 
    Translates an list of adb getevents to adb sendevent for playback
    E.g.
       Input  : /dev/input/event0: 0001 00e5 00000001
       Output : sendevent /dev/input/event0 1 229 1
